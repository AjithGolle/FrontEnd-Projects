Python

1. Data Types

1.1 primitive

  1. String : "Hello"
  2. Integer : ..,-1,0,1,..
  3. Float : 1.2,2.3,3.4
  4. Boolean : True or False
  5. Complex : i+3j,...

1.2 Non-Primitive

  1. List : [1, 2, 3, 4]
  2. Tuple : (10.0, 20.0)
  3. Set : {1, 2, 3, 4}
  4. Dictionary : {"name": "Alice", "age": 30}

2. Operators

  1. Arithmetic Operators : +, -, *, /, //, %
  2. Assignment Operators : +=, -=, *=, /=, %=, **=, //=
  3. Comparision Operators : ==, !=, >, <, >=, <=
  4. Logical Operators : and, or, not
  5. Bitwise Operators : &, |, ^, ~, <<, >>
  6. Membership Operators : in, not in

  1. if Statement
  2. if-else Statement
  3. if-elif-else Statements
  4. Nested if Statements
